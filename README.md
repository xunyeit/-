讯业软件开发/讯业代客泊车-java代客泊车系统-客户端/泊车师傅端
一、产品概述
软件功能工作流程：围绕机场代客泊车，医院代客泊车，为服务代替客户泊车预约下单的一个小程序，分为用户端和泊车师傅端，两个手机端口。

二、所有端口
用户端：微信小程序
泊车师傅端：微信小程序
后台总管理端：PC网页


三、功能明细
系统分销逻辑：三级分销（返积分，积分可抵现）

赠送logo设计

微信授权登录：获取用户手机号

首页：

1.轮播图

2.时间日期

3.代客泊车：（输入航班，输入车牌号，输入取车日期，输入提车日期，点击支付）

1.机场泊车。2.高铁站泊车。3.医院泊车。

细节：图片+标题+累计预约人数+单价

细节：图片+标题+单价+费用说明+增值服务（只展示没有功能）

点击立即预约，显示预约，后台受理后，接收到短信验证码。

退款流程：在后台没有受理订单，用户没有收到短信之前，用户可以点击取消订单，金额原路返回

费用计算方式：泊车服务费（固定）+停车费（按天算）

取车流程：用户点击订单取车，出现5位数核销码，泊车师傅端，输入核销码，核销成功，本单结束

逾期流程：用户逾期，比如下单取车日期是5号，取车时候是7号的情况。

情况一：用户自觉修改订单时间，补订单差价

情况二：用户不自觉说出取车码，泊车师傅核销系统提示，逾期，用户修改时间，系统重新出码，泊车师傅进行核销

4.自助泊车：（属于自己的停车场，显示停车场的位置，选择停放时间，进行线上支付即可）

汽车洗护：显示洗车项目和价格，点击进行购买，后台收到订单

机场贵宾厅：无功能，点击显示敬请期待

捷易登机：无功能，点击显示敬请期待

快速安检：无功能，点击显示敬请期待

企业合作：

展示图片文字介绍，点击合作咨询出手机号

个人中心：

1.头像+昵称+手机号

2.订单：展示代客泊车订单，自助泊车订单，洗车订单

3.积分：积分可以抵扣下单金额，1积分1元钱

4.我的积分：展示自己获得拉新明细（分销逻辑分销，返佣是返的积分，后台可调比例）

5.关于平台：显示平台图片介绍

6.在线客服：对接微信小程序客服

7.服务协议：展示服务条款

8.设置：修改个人信息

泊车师傅端：

小程序账号后台开通泊车师傅端，在小程序个人中心出现泊车师傅端按钮点击进入切换身份

核销订单：输入核销码，核销取车订单

订单：展示给自己指派的订单，服务的状态，待接单/服务中/已完成

我的：展示自己个人信息。展示自己接单数，自己接单金额，自己的接单明细
泊车后需要拍照车多方位照片，和视频，上传，客户进行订单页面能查看

后台管理：

用户管理

泊车师傅管理

派单大厅：订单来了之后，待受理，服务中，已完成

财务明细

航班接口：对接第三方航班信息，用户的航班信息可以同步过来

分佣比例调整


延误逻辑附加：因为按天来计算，凌晨为节点，延误4个小时以内按照3元一小时来补差价，超过4小时，按照1天来补差价

短信提醒功能：新订单下到后台，客服手机收到一条短信。客服点击受理此订单，用户收到一条短信。指派给泊车师傅，泊车师傅收到一条短信
